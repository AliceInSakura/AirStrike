# 🎮 AirStrike

> 바다 위 창공에서 펼쳐지는 공대공 3인칭 전투기 슈팅 게임

---

## 📌 게임 설명

**AirStrike**는 기본적인 전투기 조작과 전투기 HUD를 구현한 3인칭 항공 슈팅 게임입니다.  
플레이어는 전투기를 조종하여 격추되기 전까지 최대한 많은 적기를 격추해야 합니다.
